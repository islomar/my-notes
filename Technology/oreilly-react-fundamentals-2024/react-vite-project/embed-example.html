React App Embedding Example

Note: For this example to work, you need to have your React application running locally at http://localhost:5173

Start your application with: npm run dev

Embedding using an iframe

How to use this example

  1. Start your React application with npm run dev
  2. Open this HTML file in a browser
  3. You should see your React application embedded in the iframe above

For production use, you would replace the localhost URL with your deployed application URL.

See the run_and_embed_instructions.md file for more embedding options.
